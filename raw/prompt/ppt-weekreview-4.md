## 三、各章节页面布局与内容DNA

### 第一章：聚焦1个目标——收入

#### 1. 销售收入
- **页面分布**：上下分层结构
- **上半区**：
  - **文字概述**：1-2行文字总结当期收入完成情况（时间区间、全州完成金额、任务完成率、排名靠前/靠后区县、环比变化趋势）
  - **核心表格**：各区县销售收入总表
    - *列头*：区县（花垣、保靖、永顺、龙山、泸溪、凤凰、古丈、吉首、全州）
    - *行头*：销售收入周目标、累计完成、完成率、排名、环比上周、其中线下销售收入、线下环比上周
- **下半区**：左右分栏（左完成率、右环比变化）
  - **下左**：
    - **文字点评**：1-2行文字点评各区县细项指标完成率（正向指标：新增、升档、办包、初装费、调测费；负向指标：降档），指出落后进度项及落后区县
    - **表格DNA**：细项指标完成率表（正向5项+负向1项）
  - **下右**：
    - **文字点评**：1-2行文字点评各区县细项指标环比上周变化（初装费/升档/新增下降，办包/调测费提升，古丈各项指标下降）
    - **表格DNA**：细项指标环比上周情况表（正向5项+负向1项，含环比百分比）

#### 2. 2+8路径完成情况
- **页面分布**：上下分层结构
- **上半区**：
  - **文字概述**：1-2行文字总结2+8路径增收完成情况（完成金额、任务完成率、排名、环比提升幅度）
  - **核心表格**：各区县2+8路径完成进度表
    - *列头*：区县
    - *行头*：2+8路径周增收目标、周完成、完成率、排名、环比上周
- **下半区**：左右或上下分栏
  - **下左/下区**：
    - **文字点评**：1-2行文字点评各细项指标完成进度（FTTR未破零、云电脑/大屏包/爱家服务/特色卡品落后、各区县短板）
    - **表格DNA**：2+8收入路径完成进度表
      - *行头*：2项（存量降档控损、新增客户创收）+ 8项（FTTR直降礼包、爱家清洗服务、云电脑、守机宝、大屏包、特色卡品/益丰联名卡、健康宝、甄选会员）
      - *数据特征*：完成率百分比，0%表示未破零
  - **下右/下区**：
    - **文字点评**：1-2行文字点评环比变化（新增/FTTR/大屏包下降，龙山/保靖/永顺下降指标多）
    - **表格DNA**：2+8收入路径周环比变化表（同上10项指标，含环比百分比，负值标红逻辑）

### 第二章：做大两个规模

#### 1. 新增放号及份额
- **页面分布**：上下分层结构
- **上半区**：
  - **文字概述**：1-2行文字概述放号产能（日均放号量、完成率、产能评价、出账/通话净增情况）
  - **竞品表格**：DX日新增量 vs LT日新增量（7天数据对比）
- **下半区**：单一大表或左右分栏
  - **表格DNA**：客户规模与质量综合表
    - *区块1-出账/通话净增*：当月净增（较上月月底）
    - *区块2-新增通话份额*：新增份额、环比上周
    - *区块3-新增规模*：放号规模（户）、完成率、环比上周；139+大套、完成率、环比上周
    - *区块4-质量管控*：纯新增占比、零低预存占比、入网即通信占比
  - **文字点评**：嵌入在表格上方或下方，指出8个区县共性/个性问题（如古丈139+大套低于50%且环比下降超40%）

#### 2. 新增放号产能路径
- **页面分布**：左右双栏或上下双区（拉新 vs 拉大套）
- **左区/上区：拉新路径**
  - **文字概述**：分渠道、载体、场景三段式点评
  - **表格DNA1-渠道拉新**：实体渠道（目标/完成/完成率）、直销、二级放号平台
  - **表格DNA2-载体拉新**：终端（含顺差）、宽带、智家（含FTTR）——均含指导目标值、完成值、完成率
  - **表格DNA3-场景拉新**：农村场景、商聚场景、集团单位、区域客群——均含指导目标值、完成值、完成率
  - **汇总行**：后2短板数量汇总
- **右区/下区：拉大套路径**
  - **文字概述**：分渠道、载体、场景三段式点评
  - **表格DNA**：与拉新同构，但指标为"拉大套"（渠道[实体/直销]、载体[终端/宽带/智家]、场景[农村/商聚/集团/区域客群]）
  - **汇总行**：后2短板数量汇总

#### 3. 宽带新增及净增
- **页面分布**：上下结构
- **上半区**：
  - **文字概述**：1-2行文字总结全州宽带净增/新增/离网情况（净增户数、目标完成率、环比变化；新增户数、环比；离网户数、环比）
- **下半区**：
  - **超宽表格DNA**：三区段并列（净增、新增、离网）
    - *净增段*：规模、目标完成、环比变化、其中家宽、环比变化、其中企宽、环比变化
    - *新增段*：规模、环比变化、其中家宽、环比变化、其中全球通、环比变化、其中企宽、环比变化
    - *离网段*：规模、环比变化、其中家宽、环比变化、其中企宽、环比变化
  - **文字点评**：嵌入表前，指出凤凰/古丈新增下降、古丈/保靖/永顺/泸溪家宽负增长等

## 四、核心表格DNA共性特征

| 特征维度 | DNA描述 |
|---------|---------|
| **区县维度** | 固定8个区县（花垣、保靖、永顺、龙山、泸溪、凤凰、古丈、吉首）+ 全州汇总 |
| **时间口径** | 当周（4月9日-15日或4月7日-14日）+ 环比上周 |
| **指标分级** | 一级指标（收入/客户/宽带）→ 二级路径（2+8/拉新&拉大套/渠道&载体&场景）→ 三级细项（新增/升档/办包/初装费等） |
| **条件格式** | 完成率：低于50%或排名后三位需标红关注；环比：负增长标红，正增长标绿；排名：后两位标红 |
| **点评模式** | "从...看，...靠前/靠后；从环比看，...提升/下滑；请重点关注..." |
| **短板汇总** | 部分页面底部设"后2短板数量汇总"行，统计各区县落后指标个数 |
