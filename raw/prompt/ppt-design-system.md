## 一、页面基础设置

| 项目 | 规格 |
|------|------|
| **幻灯片比例** | 16:9 宽屏（标准） |
| **页面宽度** | 13.333 英寸（33.867 cm / 12192000 EMU） |
| **页面高度** | 7.500 英寸（19.050 cm / 6858000 EMU） |
| **安全边距（上）** | 0.40 in |
| **安全边距（下）** | 0.40 in |
| **安全边距（左）** | 0.40 in |
| **安全边距（右）** | 0.40 in |
| **内容区宽度** | 12.53 in（页宽减去左右边距） |
| **主内容区高度** | 6.30 in（标题栏以下区域） |
| **背景色** | 纯白 `#FFFFFF` 或浅灰 `#F5F6F8` |
| **分辨率** | 导出 PNG 不低于 150 DPI，正式版不低于 300 DPI |

## 二、色彩体系

### 2.1 主色调（品牌蓝）

> 内部分析报告主色多为中性深蓝/藏蓝，区别于市场PPT的鲜艳蓝。

| 色彩名称 | 用途 | RGB | HEX |
|----------|------|-----|-----|
| **主题蓝（深）** | 页眉、主标题、关键数字高亮 | (0, 70, 127) | `#00467F` |
| **主题蓝（中）** | 二级标题、图表主色 | (0, 112, 192) | `#0070C0` |
| **主题蓝（浅）** | 表格条纹、背景填充 | (214, 231, 245) | `#D6E7F5` |
| **品牌蓝（辅助）** | 可视化辅助系列 | (65, 143, 222) | `#418FDE` |

### 2.2 中性色（灰阶）

| 色彩名称 | 用途 | HEX |
|----------|------|-----|
| **深灰（正文）** | 正文主色，比纯黑更柔和 | `#1A1A1A` |
| **中灰（辅助文字）** | 注释、标注、来源说明 | `#595959` |
| **浅灰（弱信息）** | 禁用状态、背景分割线 | `#8C8C8C` |
| **边框灰** | 表格线、分隔线 | `#D9D9D9` |
| **背景灰** | 卡片背景、区块底色 | `#F5F6F8` |
| **纯白** | 页面背景、文字反白 | `#FFFFFF` |

### 2.3 语义色（数据状态）

## 三、字体与字号规范

### 3.1 字体优先级

| 优先级 | 字体名称 | 适用场景 |
|--------|----------|----------|
| **首选** | 思源黑体 / Source Han Sans CN | 支持思源时优先，视觉一致性最好 |
| **次选** | 微软雅黑（Microsoft YaHei） | Windows 环境默认，最广兼容 |
| **数字专用** | DIN Next / Barlow / Roboto | 纯数字、KPI 大字、百分比展示 |
| **英文/代码** | Arial / Helvetica Neue | 英文术语、缩写、代码片段 |
| **禁用** | 宋体、仿宋、楷体、黑体（非思源） | 屏幕阅读效果差，形象不专业 |

### 3.2 字号体系

| 层级 | 应用场景 | 字号 | 字重 |
|------|----------|------|------|
| **D0** | 封面主标题 | 36–40 pt | Bold |
| **D1** | 页面主标题（一级标题） | 22–24 pt | Bold |
| **D2** | 模块标题（二级标题） | 16–18 pt | SemiBold |
| **D3** | 小节标题（三级标题） | 14–15 pt | SemiBold |
| **B1** | 正文主体 | 13–14 pt | Regular / Medium |
| **B2** | 辅助说明、标注 | 11–12 pt | Regular |
| **B3** | 来源、注释、页脚 | 9–10 pt | Regular（浅灰色） |
| **KPI** | 大数字展示（KPI卡片） | 32–48 pt | Bold |
| **TAG** | 标签、角标 | 9–11 pt | Medium |

|------|------|

| **字重** | Bold |

| **对齐** | 左对齐 |
| **行距** | 1.0 倍 |
| **装饰** | 左侧可加 3–4 pt 竖线色块（`#0070C0`），高度与文字行高一致 |

| **对齐** | 左对齐 |

| **装饰** | 可在标题下方添加 1 pt 横线（`#D9D9D9`） 或背景色条 |

## 五、正文文本规范

### 5.1 核心摘要文字（Summary Box）

> 每页顶部 1–2 行核心结论，浓缩本页最重要信息。

| 属性 | 规格 |
|------|------|
| **字体** | 微软雅黑 |
| **字号** | 13–14 pt |
| **字重** | Regular 或 Medium |
| **颜色** | `#1A1A1A` |
| **行距** | 1.5–1.7 倍 |
| **背景** | 浅蓝背景条 `#D6E7F5` 或左侧 3pt 蓝色竖线 + 无背景 |
| **内边距** | 上下 6 pt，左右 10 pt |
| **位置** | 紧接主标题下方，高度约 0.6–1.0 in |

### 5.2 常规正文（B1）

| 属性 | 规格 |
|------|------|
| **字体** | 微软雅黑 |
| **字号** | 13–14 pt（旧版参考：14 pt） |
| **样式** | Regular / Medium / Bold（旧版参考：加粗） |
| **颜色** | `#1A1A1A`（深灰，旧版参考：`#000000` 黑色） |
| **对齐方式** | 左对齐 |
| **行距** | 1.4 倍（长文本）/ 1.7 倍（摘要文本） |
| **段前/段后距** | 0 pt |
| **文本框边距** | 0 in（无边距）或 左/右 0.1 in，上/下 0.05 in |

### 5.3 正文列表

| 属性 | 规格 |
|------|------|
| **字体** | 微软雅黑 |
| **字号** | 12–13 pt |
| **字重** | Regular |
| **颜色** | `#1A1A1A` |
| **行距** | 1.4 倍 |
| **列表符号** | "•" 圆点（推荐），或数字序号，不使用 PPT 默认样式 |
| **缩进** | 一级列表左缩进 0.2 in，二级缩进 0.4 in |

### 5.4 数据标注 / 辅助说明（B2）

| 属性 | 规格 |
|------|------|
| **字体** | 微软雅黑 |
| **字号** | 11–12 pt（旧版参考：12 pt） |
| **字重** | Regular / Bold（旧版参考：加粗） |
| **颜色** | 正向：`#1AAB55`（旧版参考：`#00B050` 亮绿），负向：`#CF1322`（旧版参考：`#C00000` 亮红），中性：`#595959` |
| **符号** | 增长用 "▲"，下降用 "▼"，与数字同色 |

### 5.5 来源 / 注释文字（B3）

| 属性 | 规格 |
|------|------|
| **字体** | 微软雅黑 |
| **字号** | 9–10 pt（旧版参考：10 pt） |
| **字重** | Regular / Bold（旧版参考：加粗） |
| **颜色** | `#8C8C8C`（浅灰，旧版参考：主题色/灰色） |
| **对齐** | 左对齐 |
| **位置** | 页面底部，距底边 0.15–0.25 in |
| **格式** | "注：……" 或 "数据来源：……" |

## 六、图表与数据可视化规范

### 6.1 通用图表原则

- **数据墨水比**：去除所有非必要的网格线、刻度线、图例边框
- **图表标题**：简短，说明"什么数据/时间范围"，15 pt Bold
- **数轴标签**：11 pt，中灰 `#595959`
- **数据标签**：关键数据直接在图上显示，12 pt，与系列色一致
- **图例**：放在图表顶部或右侧，不放底部（避免与注释混淆）
- **基准线 / 目标线**：虚线表示，颜色 `#8C8C8C`，2 pt 线宽

### 6.2 柱状图 / 条形图

| 属性 | 规格 |
|------|------|
| **系列宽度** | 60–70%（GAP 设置 30–40%） |
| **分组间距** | 组内间距 5–8% |
| **边框** | 无边框 |
| **数据标签位置** | 柱顶外（正值）/ 柱底外（负值） |
| **背景** | 无背景网格 或 浅灰横线（0.5 pt，`#E8E8E8`） |
| **系列色** | 优先使用主题蓝 `#0070C0`，多系列按色板顺序 |

### 6.3 折线图

| 属性 | 规格 |
|------|------|
| **线宽** | 2.0–2.5 pt |
| **数据点** | 圆形，直径 6–8 pt，与线同色 |
| **区域填充** | 不超过 2 个系列时可使用面积图，透明度 20–30% |
| **数轴** | 仅保留 Y 轴，X 轴用刻度替代 |
| **系列色** | 优先使用主题蓝 `#0070C0`，对比系列用暖橙 `#FA8C16` |

### 6.4 饼图 / 环形图

| 属性 | 规格 |
|------|------|
| **适用场景** | 仅用于展示占比（≤6 个分类），禁用于趋势对比 |
| **起始角度** | 12 点方向（顶部） |
| **标签** | 直接标注百分比+分类名，不用图例 |
| **最大分类** | 超过 6 个时，合并为"其他" |
| **环形内径** | 60–70%（环形图），中央可写总量 |
| **系列色** | 按色板顺序取色，最大分类使用主题蓝 |

### 6.5 散点图 / 气泡图

| 属性 | 规格 |
|------|------|
| **散点大小** | 均匀 8–12 pt（气泡图除外） |
| **参考象限线** | 中位数或均值，虚线 `#8C8C8C` |
| **标注** | 关键数据点用箭头+文字标出，避免标注过密 |

### 6.6 组合图（柱状+折线）

| 属性 | 规格 |
|------|------|
| **柱系列** | 使用主题蓝 `#0070C0` 或浅蓝 `#D6E7F5` |
| **线系列** | 使用暖橙 `#FA8C16` 或深红 `#CF1322`（用于目标/预警） |
| **主次轴** | 柱用主轴，线用次轴，次轴刻度颜色与线同色 |
| **标签** | 柱顶显示数值，线旁显示关键点 |

### 6.7 KPI 卡片样式

```
+----------------------------------+
|  [指标名称]               [环比]  |  <- 12pt 灰色 | 12pt 绿色/红色（▲▼）
|                                   |
|     32.1%                         |  <- 40pt Bold 深蓝数字
|  ─────────────────────            |  <- 分隔线
|  目标：30% | 同比：+5.2%          |  <- 11pt 灰色
+----------------------------------+
```

| 属性 | 规格 |
|------|------|
| **卡片背景** | 白色 `#FFFFFF` 或极浅蓝 `#F0F7FF` |
| **卡片边框** | 1 pt `#D9D9D9` 或无边框（使用阴影代替） |
| **圆角** | 4–6 pt |
| **阴影** | 可选，模糊 4–6 pt，透明度 15–20% |
| **KPI数字** | 36–48 pt，Bold，`#00467F` 或根据状态变色 |
| **指标名** | 12–13 pt，`#595959` |
| **环比标注** | 12 pt，▲绿 `#1AAB55` / ▼红 `#CF1322` |

### 6.8 瀑布图

| 属性 | 规格 |
|------|------|
| **适用场景** | 展示累计构成、增减变化过程（如收入拆解、成本构成） |
| **起始柱** | 使用主题蓝 `#0070C0` |
| **正向增长柱** | 增长绿 `#1AAB55` |
| **负向下降柱** | 警示红 `#CF1322` |
| **汇总柱** | 深蓝 `#00467F` 或中灰 `#595959` |
| **连接线** | 1 pt 虚线，`#8C8C8C` |
| **数据标签** | 柱顶/柱底居中显示，11–12 pt，白色或深色（根据柱色对比度） |
| **背景** | 无网格线，仅保留水平基准线 |
| **图例** | 必须包含，说明各颜色含义 |

### 6.9 柱状折线图（双轴）

| 属性 | 规格 |
|------|------|
| **柱系列** | 使用主题蓝 `#0070C0` 或浅蓝 `#D6E7F5`，柱宽 60–70% |
| **折线系列** | 使用暖橙 `#FA8C16` 或深红 `#CF1322`，线宽 2.5 pt |
| **主次轴** | 柱用主轴（左侧），线用次轴（右侧） |
| **次轴刻度** | 颜色与折线系列一致 |
| **数据标签** | 柱顶显示数值（12 pt，深蓝）；折线关键点旁标注（12 pt，与线同色） |
| **数据点** | 圆形，直径 6–8 pt，与折线同色，白色描边 1 pt |
| **背景** | 无背景网格 或 浅灰横线（0.5 pt，`#E8E8E8`） |
| **图例** | 顶部或右侧，说明柱与线分别代表什么指标 |

### 6.10 环形饼图

| 属性 | 规格 |
|------|------|
| **适用场景** | 占比展示（≤6 个分类），中央可放置总计或核心指标 |
| **环形内径** | 60–70%（外径 100%） |
| **起始角度** | 12 点方向（顶部） |
| **系列色** | 按色板顺序取色，最大分类使用主题蓝 `#0070C0` |
| **最大分类** | 超过 6 个时，合并为"其他" |
| **数据标签** | 直接标注百分比+分类名，11–12 pt，与扇区同色或深灰 `#1A1A1A` |
| **图例** | 不使用图例（直接标签替代） |
| **中央文字** | 可放置总量（KPI 样式，24 pt Bold，`#00467F`）或指标名称（12 pt，`#595959`） |
| **边框** | 扇区间可保留 1 pt 白色分隔线，增强边界感 |

## 七、表格样式规范

### 7.1 表格整体风格

> 采用"无外框线 + 内部横线"简洁风，对标麦肯锡/头部大厂报告表格。

| 属性 | 规格 |
|------|------|
| **外边框** | 无（None） |
| **内部竖线** | 无（None） |
| **内部横线** | 0.5–1 pt，`#D9D9D9`（浅灰） |
| **表头底线** | 1.5–2 pt，`#00467F`（主题深蓝） |

### 7.2 表头行

## 十三、排版原则与禁止事项

### 13.1 核心排版原则

1. **结论前置**：每页顶部摘要条须给出明确结论，而非描述性陈述
   - 正确："本月 DAU 同比增长 18%，超目标 5pct，主要来自新用户拉新贡献"
   - 错误："以下为本月 DAU 数据分析"

2. **层级分明**：通过字号和字重建立清晰视觉层级（不超过 3 个层级）
   - 主标题 → 模块标题 → 正文，三层足够

3. **密度适中**：每页文字量不超过 200 字（不含表格），图表留白 >= 15%

4. **色彩语义化**：
   - 蓝色：标题、主题、中性信息
   - 绿色：正向、达标、增长
   - 红色：负向、预警、下降
   - 橙色：关注、待确认、风险提示

5. **对齐严格**：所有元素遵循网格对齐，不允许随意偏移
   - 文字元素统一 0.40 in 左边距
   - 图表/表格之间间距 >= 0.15 in

6. **留白适度**：文本框边距保持 0.05–0.1 in，避免文字贴边；图表与文字之间保持适当呼吸感

7. **字体统一**：全篇仅使用 1–2 种字体，数字可单独使用数字专用字体

### 13.2 禁止事项

| 禁止项 | 原因 |
|--------|------|
| 使用超过 4 种颜色（图表系列色除外） | 视觉混乱，失去专业感 |
| 同一页使用多种字体 | 显得不统一 |
| 字号低于 9 pt | 投影/屏幕不可读 |
| 纯黑 `#000000` 作为正文色 | 刺眼，大厂普遍使用深灰 `#1A1A1A` |
| 图表无标题 | 读者无法快速理解 |
| 饼图展示趋势数据 | 饼图只适合占比展示 |
| 标题只写描述不写结论 | 内部报告要直接给出 insight |
| 使用渐变色填充柱子 | 导致打印失真，视觉干扰 |
| 阴影过重（透明度<80%） | 显得廉价 |
| 背景用图片或纹理 | 严重干扰数据阅读 |
| 全文使用加粗 | 失去强调意义，无主次之分 |
| 多行全大写中文 | 严重影响可读性 |

### 13.3 快速检查清单

在发送报告前，逐项确认：

- [ ] 每页是否有明确结论（不只是描述）？
- [ ] 主标题字号是否统一？
- [ ] 颜色使用是否符合语义（红/绿/蓝用法正确）？
- [ ] 图表是否有标题和数据来源？
- [ ] 所有元素是否对齐到网格？
- [ ] 数字字号是否能在投影仪上清晰阅读（>=11 pt）？
- [ ] 是否有多余的装饰、边框、阴影影响数据阅读？
- [ ] 页脚是否标注数据来源和日期？
- [ ] 全篇字体是否一致？
- [ ] 打印为黑白后是否仍可区分关键数据？

## 十四、图表美化与可视化布局

### 14.1 可视化原则

- **数据墨水比**：去除所有非必要的网格线、刻度线、图例边框，让数据本身成为视觉焦点
- **去除冗余**：删除重复标签、过多图例、无意义装饰线
- **一图一结论**：每张图表只传达一个核心结论，避免信息过载
- **视觉层次**：通过颜色深浅、字号大小建立数据主次关系

#### 14.3.6 响应式与打印适配

- 图表容器使用相对尺寸（百分比或 flex），避免固定像素宽度

### 14.4 图表选型指南

| 数据目的 | 推荐图表类型 | 替代方案 | 禁用方案 |
|----------|--------------|----------|----------|
| **对比** | 柱状图 / 条形图 | 分组柱状图、堆叠柱状图 | 饼图、折线图 |
| **趋势** | 折线图 / 面积图 | 阶梯图、平滑曲线图 | 柱状图、饼图 |
| **占比** | 饼图 / 环形图 / Treemap | 百分比堆叠柱状图 | 折线图、散点图 |
| **构成** | 瀑布图 / 堆叠柱状图 | 瀑布图（展示增减过程） | 饼图（超过 6 项时） |
| **关系** | 散点图 / 气泡图 | 四象限矩阵图 | 折线图、柱状图 |
| **多维** | 雷达图 | 平行坐标图 | 饼图 |
| **双指标** | 柱状折线图（双轴） | 双轴图 | 单轴强行叠加 |

### 14.5 数据转可视化检查清单

在将表格数据转为图表前，逐项确认：

- [ ] **数据是否适合该图表类型**：时间序列不用饼图，占比不用折线图
- [ ] **颜色是否符合语义**：绿色代表正向/达标/增长，红色代表负向/预警/下降，蓝色代表中性/主题
- [ ] **关键数据是否有直接标注**：TOP 值、异常值、目标值是否在图上直接标出
- [ ] **图例是否清晰说明系列含义**：避免使用 "系列1"、"系列2" 等无意义名称
- [ ] **坐标轴是否从 0 开始**（柱状图必须），折线图可根据数据范围适当调整
- [ ] **是否有目标线/基准线**：完成率类图表必须标注 100% 目标线
- [ ] **数据标签是否重叠**：密集数据可省略标签，改用提示框展示
- [ ] **打印/黑白模式是否可区分**：多系列图表在线条样式或纹理上需有区分度
