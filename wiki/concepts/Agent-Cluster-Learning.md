## Definition

Agent集群学习法是指使用多个专业化AI Agent（而非单一通用AI）组成协作团队，模拟真实世界的专业分工，以解决复杂学习问题的教育方法。每个Agent在各自专精领域深度输出，通过主控Agent统筹协调。

## Core Mechanism

### 架构

```
┌─────────────────────────────────────┐
│     主控Agent：价值投资教练          │
│     （总指挥·每日上课·心智纠偏）      │
└──────────────┬──────────────────────┘
               │
    ┌──────────┼──────────┐
    ▼          ▼          ▼
┌────────┐ ┌────────┐ ┌────────┐
│财报解剖师│ │案例图书馆│ │心性训练师│
│（数据层）│ │（认知层）│ │（心理层）│
└────────┘ └────────┘ └────┬───┘
                            │
                            ▼
                     ┌────────┐
                     │书单管理员│
                     │（资源层）│
                     └────────┘
```

### 协作节奏

| 时间 | 活动 | Agent |
|------|------|-------|
| 周一 | 发布阅读计划 | 书单管理员 |
| 周二-周五 | 每日一课 | 主控Agent |
| 周六 | 专题深度学习 | 财报/案例 |
| 周日 | 压力测试+复盘 | 心性+主控 |

### vs 传统自学

| 维度 | 传统自学 | Agent集群 |
|------|---------|----------|
| 专业化 | 一本书一个视角 | 五位专家各管一摊 |
| 记忆 | 读完就忘 | 完整记录学习轨迹 |
| 反馈 | 3年后市场惩罚 | 3分钟内纠错 |

## Applications

- 价值投资学习（本手册原型）
- 任何需要「理论+实践+反馈」的复杂技能
